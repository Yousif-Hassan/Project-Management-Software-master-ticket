How To stap voor stap
Creeer projecten:
  1. Ga naar /home page
  2. klik icoon rechtsbovenin
  3. typ het naam van het project (moet 3 of meer symbolen bevatten, aleen A-Za-z0-9_)
  4. De afkorting wordt automatisch geschreven, als u wilt, kunt u deze niet wijzigen. Alleen letters, 3 symbolen
  5. Click om een button aan te maken
wijzig projecten:

Projecten kunnen worden bewerkt vanaf / home page en vanuit / settings (daar moet je beheerder zijn)

  • Bewerken vanaf de startpagina (voor niet-beheerders)
    1. ga naar /home page
    2. klik iccon rechtsbovenin
    3. kies het gewenste project
    4. verander naam of afkortingen
    5. klik button aanmaken
  • Bewerken vanaf de instellingenpagina (alleen voor beheerders)
    1. Ga anar /settings pagina (vanaf /home en icoon rechtsbovenin)
    2. Ga naar Projecten link vanaf linkerkant van menu
    3. daar kan je projecten toevoegen, verwijderen of bewerken
Selecteren van projecten:
  1. Ga naar /home page
  2. Kies uit het geselecteerde menu het benodigde project
  3. daar kan je projecten toevoegen, verwijderen of bewerken
  4. onderstaande links leiden naar tickets / wiki van dit project
Creer nieuwe ticket:
  1. Ga naar /home page
  2. Kies uit het geselecteerde menu het benodigde project
  3. Klik ticket link beneden
  4. Ga naar Nieuwe probleem Button
  5. type in de benodigde velden (*)
  6. klik Create button
wijzig ticket:
  1. Ga naar ticket preview
  2. Klik linksboven op het ticket op de knop Bewerken
  3. maak je wijziging
  4. klikupdate button
verander ticket status:

Ticketstatussen kunnen worden gewijzigd vanaf het dashboard met 'slepen', vanuit ticketvoorbeeld of bij het bewerken van ticket

Gebruik van volgsysteem
  1. ga naar preview ticket
  2. Aan de rechterkant bij resolutie> 767 en onderaan bij resolutie 767 is het volgsysteem
  3. vanaf het volgen van knoppen wordt gestart en de timer voor dit ticket gaat bovenaan de pagina bij het vernieuwen
  4. tijd laagste van 1 minuut kan niet worden gered
  5. Vanaf button kan je tijd toevoegen
Creeer wiki ruimte
  1. Ga naar /home page
  2. Kies uit het geselecteerde menu het benodigde project
  3. Klik wiki link beneden
  4. Klik Spaces lijst aan de boven kant
  5. Klik link aan de onderkant Creeer nieuw eentje
  6. Typ benodigde velden
  7. Klik Creeer button
Creeer wiki pagina
  1. Ga naar /Home pagina
  2. Kies uit het geselecteerde menu het benodigde project
  3. klik wiki link beneden
  4. KLik Creeer button aan de bovenkant van de pagina
  5. kies ruimte waar de project naartoe gaat
  6. Typ de bovenliggende pagina voor uw pagina (uw pagina wordt een onderliggende pagina van ..) (niet vereist)
  7. Kies template
  8. Klik Volgende button
  9. Typ naam in je pagina
  10. Klik Create en update je content pagina als het nodig is
Creeer wiki template
  1. Ga naar /home pagina
  2. Ga naar/ settings pagina (vanaf /home en icoon rechtsbovenin)
  3. klik Wiki Templates vanaf rechterzijde van het menu
  4. beheer je templates
beheer users
  1. Ga naar / home pagina
  2. Ga naar / settings pagina (vanaf /home en icoon rechtsbovenin)
  3. Klik Profielen vanaf rechterzijde van het menu
  4. Beheer je gebruikers
verander standaar taal Je hebt de mogelijkheid om het voor elke gebruiker afzonderlijk te wijzigen!
  1. Ga naar / settings
  2. Open 'general' pagina
  3. Selecteer je taal
  4. opslaan

Debug Mode

Het systeem heeft een foutopsporingsmodus. Wanneer php constant DEBUG_MODE in /inc/settings.php is geplaatst naar TRUE onder aan de pagina worden alle mysql-queries weergegeven. Omleidingen werken niet en php herneemt alle fouten!

Multi taal

Om een ​​nieuwe taal te maken, moet u bestanden maken .js and .php in /lang directory in het vereiste formaat (u kunt het in de originele bestanden zien).

The array(property) lang_php in Class Main keeps all words in lang_ABBR.php file. Words in array are called from files. lang_ABBR.js is loaded from header and words are pasted in javascripts from loaded object.

ABBR-naam in js- en php-bestanden moet in HOOFDLETTERS zijn!

Sync email account

Om een ​​e-mailaccount te synchroniseren, stelt u gewoon parameters in op de instellingenpagina of wanneer u een project maakt vanaf de startpagina

Om e-mails van een e-mailaccount te kunnen vinden, moet dit zijn ingesteld cron job voor bestand crons/fetch_imap.php. Voor elke oproep zal fetch_imap alle projecten herhalen en hun connecties krijgen en e-mails voor hem ophalen.

E-mails ophalen is met php / imap, e-mails verzenden is met PHPMailer Klasse, hij wordt geladen vanuit de methode sendEmal in de hoofdklasse, daar kun je de opties van PHPMailer beheren.
